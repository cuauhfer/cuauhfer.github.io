Inovag App

Es una versión de Front end que sirvió como base para la creación de las funciones y servicios API para la puesta en marcha de la aplicación móvil de Inovag.

Inovag App

Descripción

La aplicación cuenta con una pantalla para el inicio de sesión de los clientes, al conectarse al servidor remoto, obtiene los datos e información del usuario, mostrando sus cuentas y servicios adquiridos con la empresa, al acceder a cada uno de estos servicios, se pueden visualizar el saldo, movimientos y detalles de dicha cuenta. De igual manera, la aplicación permite acceder a funciones como llamadas y contacto con la empresa.
