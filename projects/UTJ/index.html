Sistema web desarrollado para realizar la evaluación interna de los programas educativos de la UTJ.

Sistema de Pre-Evaluación de Carreras Educativas

Descripción

El sistema comprende una serie de páginas y formularios los cuales permiten a la institución realizar la pre-evaluación interna de sus programas educativos, este proceso comprende una autoevaluación por parte de los directores de división, una revisión externa, visita y revisión a las instalaciones y la elaboración de un dictamen final.
